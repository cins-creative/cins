01Typography Designer là ai?

Typography Designer là chuyên gia design + draw typeface character set. Phân loại theo focus: (1) Type Designer — design typeface full character set (Latin, Cyrillic, Greek, CJK); (2) Lettering Artist — custom drawn letter cho specific use (logo, poster, packaging), không scale to full font; (3) Calligraphy Artist — hand-drawn traditional; (4) Display Font specialist — large size, decorative; (5) Text Font specialist — body text, readability focus. Phân loại theo employer: (1) Type Foundry — Monotype, Linotype, Adobe Type, Hoefler&Co, House Industries, Klim Type Foundry; (2) Tech company in-house — Google (Roboto), Apple (San Francisco), Microsoft (Segoe); (3) Brand agency — custom typeface project; (4) Freelance + Indie Type Designer. Phân loại theo script: (1) Latin dominant; (2) Vietnamese diacritic specialty; (3) Cyrillic + Greek; (4) CJK (Chinese, Japanese, Korean); (5) Arabic + Hebrew + Indic.

Khác với Graphic Designer dùng font — Typography Designer make font others use. Khác với Lettering Artist — Type Designer scale to full font, Lettering one-off. Khác với Calligrapher — Type Designer digital, Calligrapher hand. Khác với Font Engineer (technical hinting + OpenType programming) — Type Designer focus form, Engineer focus tech (overlap exist). Hợp tác chặt với: Brand Designer + Creative Director (custom typeface project), Type Foundry team, Font Engineer (tech), Localization team (script expert), Linguist + Typographer scholar.

02Công việc của một Typography Designer

1. Brief + research + concept

Foundation. Brief intake: who use, what for, mood, technical requirement. Reference research: typography history book, historical specimen. Mood board: visual direction. Sketch by hand: 5-10 letter (n, o, a, e, s) define rhythm. Digital convert: scan + trace in Glyphs / FontLab. Iterate: với client / Creative Director.

  • Brief + reference
  • Hand sketch
  • Digital concept

2. Draw + design typeface — primary work

Core. Draw character set: lowercase, uppercase, number, punctuation, accent. Latin basic = 200+ glyph. Vietnamese diacritic complete = 500+ glyph. Full pro = 1,000+ glyph. Tool: Glyphs (Mac, industry standard), FontLab, Robofont. Bezier curve master: minimum point, smooth curve, consistent stress. Style variant: weight (Thin, Light, Regular, Bold, Black), italic, condensed, wide. Modern typeface = 6-12 weight × 2 italic = 12-24 file. Multi-script: Cyrillic, Greek, CJK extension.

03Typography Designer cần giỏi gì?

✒️ Letterform + form sensitivity

💾 Glyphs + FontLab + OpenType
📚 Typography history
🌐 Script + multilingual
🎯 Patience + obsession with detail
Letterform + form sensitivity

Subtle curve nuance. Stroke contrast. Stress direction. Rhythm. Read: "Thinking with Type" (Lupton), "Designing Type" (Cheng), "Letters of Credit" (Tracy).

Bezier + curve mastery

Minimum point principle. Smooth curve. Consistent across letter. Quadratic vs cubic. PostScript curve fundamentals.

Glyphs + FontLab + OpenType

Glyphs industry standard. FontLab alternative. Robofont. OpenType feature language. Python scripting in Glyphs (Robofab). Read: "How to Make a Pizza" (Glyphs tutorial).

Typography history

Roman square capital → Carolingian minuscule → Gutenberg → Garamond → Caslon → Bodoni → modern. Movement: Bauhaus, Swiss, Postmodern. Read: "The Elements of Typographic Style" (Bringhurst), "Anatomy of a Typeface" (Lawson).

Script + multilingual

Vietnamese diacritic master (huge market). Cyrillic + Greek. CJK if ambitious (huge undertaking). Arabic + Hebrew different design tradition. Read: "Bi-Scriptual" (Chahine).

Patience + obsession with detail

1,000+ glyph × multiple weight = monk-level patience. Pixel-level obsession. Long-term project (1-3 năm). Not for impatient.

Rig quality define animator efficiency + character expressiveness. Foundation technical role của 3D animation production. Without good rig, animator struggle + production slow.

01Rigging Artist hoạt hình là ai?

Rigging Artist hoạt hình là specialty của animation studio. Phân loại theo production type: (1) Feature film animation — Pixar, Disney, DreamWorks, Illumination level (long production, complex rig); (2) TV series animation — Sun Wolf, Nickelodeon, Cartoon Network (fast production, optimized rig); (3) Short film; (4) Commercial animation. Phân loại theo asset focus: (1) Hero character — main star; (2) Secondary; (3) Background / crowd; (4) Creature; (5) Vehicle + prop; (6) Facial specialty. Phân loại theo style: Photoreal CG (Pixar), Stylized cartoon (Wolfoo, Cocomelon), Hand-drawn 2D (separate workflow), Stop-motion (physical rig — armature).

Khác với VFX Rigger (live-action photoreal) — Animation Rigger stylized + animator-friendly. Khác với Game Rigger (real-time constraint, fewer joint) — Animation Rigger luxury joint count. Khác với Rigging TD (tool develop) — Rigging Artist asset-by-asset focus. Hợp tác chặt với: Modeling Artist, Animator + Animation Supervisor, Rigging TD, Director (for character feel).

2. Body rig + skeleton + control

Skeleton setup: anatomy-accurate joint. FK + IK blend hệ thống. Stretchy spine + limb: for cartoon style. Squash + stretch: principle of cartoon animation, rig built-in. Control rig: NURBS curve control intuitive. Picker UI: Maya animator UI custom. Constraint hierarchy: parent-child predictable. Custom attribute: animator slider for blend, twist, etc.

3. Facial rig + expression

Critical for character animation. Blendshape: per expression target sculpt. Hundred+ blendshape for hero (smile, frown, surprise, anger). Joint-based facial: alternative + complementary. FACS (Facial Action Coding System): scientific facial muscle base — Disney + Pixar use. Lip sync setup: phoneme blendshape (A, E, I, O, U, M, F). Eye rig: lookAt control. Custom slider: animator easy expression dial.

Anatomy + biomechanics

Human + animal anatomy. Cartoon exaggeration variant. Read: "Anatomy for the Artist" (Simblet), "Acting for Animators" (Hooks).

Animation principle

12 principle of animation. Squash + stretch, anticipation, follow-through, exaggeration. Rig built support these. Read: "The Animator's Survival Kit" (Williams), "The Illusion of Life" (Thomas + Johnston).

Math + matrix

Vector, matrix, quaternion, Euler. Joint orient. Gimbal lock avoid. Read: "3D Math Primer" (Dunn).

Maya + Houdini rigging

Maya primary. Houdini Apex (procedural rig) growing. Advanced Skeleton, mGear popular framework. Studio Library for pose. Blender alternative.

Python + MEL script

Auto-rig script. Custom tool. Python OpenMaya primary. MEL legacy. C++ for plugin (advanced).

Animator collaboration

Closest user. Listen need. Iterate. Diplomatic. Hiểu animator pain point. Many top Rigger from animator background.

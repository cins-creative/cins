Web Designer là chuyên gia website design. Phân loại theo focus: (1) Marketing website — brand site, landing page, campaign; (2) E-commerce website — shop, product detail, checkout; (3) Editorial / content website — magazine, blog, publication; (4) SaaS website — product marketing site (not app); (5) Portfolio + agency site. 03Web Designer cần giỏi gì?

Visual design + composition

Composition. Hierarchy. White space. Balance. Read: "Refactoring UI" (Schoger + Wathan), "The Principles of Beautiful Web Design" (Beaird + George).

Typography + grid

Web typography (Google Fonts, custom font). Type scale. Grid (12-col, asymmetric). Read: "On Web Typography" (Santa Maria), "Web Typography" (Hischmann).

Figma + Webflow + Framer

Figma design. Webflow + Framer visual web builder. Sketch Mac. Adobe XD. No-code + design hybrid trend.

HTML + CSS literacy

HTML semantic. CSS flexbox + grid. Responsive query. Hiểu what dev can do. Read: "HTML and CSS: Design and Build Websites" (Duckett), "Every Layout" (Andrew).

Animation + interaction

CSS animation. Scroll trigger (GSAP). Lottie. Three.js for WebGL. Read: "Designing Interface Animation" (Head).

SEO + accessibility

SEO basic (title, meta, heading, alt). WCAG accessibility. Performance literacy. Read: "SEO 2024" (McDonald), "Inclusive Design Patterns" (Pickering).

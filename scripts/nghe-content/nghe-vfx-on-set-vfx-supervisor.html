On-set VFX Supervisor là chuyên gia đại diện VFX vendor / studio on set — đảm bảo footage shot theo yêu cầu VFX. Critical role bridge production + post-production. Không có on-set VFX info, post-production có thể tốn hàng trăm thousand $ to fix scene mà supposed sẽ cheap-to-fix.

01On-set VFX Supervisor là ai?

On-set VFX Supervisor là chuyên gia VFX hiện diện trên trường quay — capture all data + reference cần thiết cho post VFX work. Phân loại theo company: (1) Studio in-house — Marvel, Lucasfilm, ILM, Weta có dedicated on-set team; (2) VFX vendor representative — major vendor (DNEG, Framestore) send supervisor cho client project; (3) Production VFX Supervisor — work for production company directly, oversee multiple vendor; (4) Indie / commercial — smaller scale operation. Phân loại theo project: Feature film, TV series, Commercial / ad, Music video.

Khác với VFX Supervisor (overall) — On-set focus only production phase; full VFX Supervisor oversee entire post pipeline. Khác với DP / Cinematographer (frame photography aesthetic) — On-set VFX focus data + technical capture. Khác với 1st AC (camera operation) — VFX role advisor + data capture. Khác với Director — supervisor execution-focus VFX side. Hợp tác chặt với: Director, DP / Cinematographer, 1st AC, Gaffer, Production Designer, Stunt Coordinator, VFX Supervisor (overall) post-side.

  • Plan VFX requirement cho mỗi shot

On-set VFX Supervisor — bridge production + post

Vị trí trong quy trình sản xuất

On-set VFX Supervisor work trên set production trong suốt principal photography (4-12 tháng cho feature). Báo cáo VFX Supervisor (overall) + Director on set. Lead small team: VFX Data Wrangler, On-set VFX Coordinator, Witness Camera Operator. Pre-production: meet với Director + DP, plan VFX shot list, prep equipment. Production: present every shooting day. Post: handoff data to post team, occasional visit.

02Công việc của một On-set VFX Supervisor

1. Pre-production planning

Trước shoot. Meet với Director, DP, Production Designer, Stunt Coordinator. Review VFX shot list + storyboard. Per VFX shot decide method: full CGI, green screen, in-camera practical, partial set extension. Test shoot — green screen lighting test, tracking marker pattern. Prep equipment: HDRI camera (Spheron / Theta Z1), color chart, witness camera, LiDAR scanner, measurement tool. Build shot tracking sheet.

  • Plan với Director + DP
  • Test shoot
  • Equipment prep

2. On-set: camera + lens data capture

Trên set per shot. Record camera + lens metadata: focal length, aperture, focus distance, ISO, shutter, sensor size. Lens distortion data — shoot calibration chart with each lens. Camera position + height + tilt — physical measurement. Sun position + time of day. All info go in shot tracking sheet — vital for post tracking + matching CGI camera.

  • Position measurement

3. Lighting + HDRI + reference

Capture HDRI panoramic (Spheron camera) of set lighting per scene — used in post for matching CGI lighting. Take color chart photo (Macbeth) for color matching. Photograph reference: 360 of set, props, costume, character makeup, lighting setup. LiDAR scan set if budget allow — point cloud for set extension / removal in post.

4. Green screen + tracking marker supervise

For green screen shot: supervise lighting (even, no spill on subject), tracking marker pattern (placed for camera tracking in post), witness camera placement (record alternate angle for triangulation). For practical-heavy shot: tracking marker placement on subject + set. Advice DP on technical requirement — sometimes change setup to make post easier. Handoff VFX data package to post team end of day.

03On-set VFX Supervisor cần giỏi gì?

🛠️ Tool: HDRI capture, LiDAR, color chart
💬 Communication on set
⏰ Pressure handle + decision speed
Camera + lens technical

Deep camera knowledge: ARRI Alexa, RED, Sony Venice — each system specification. Lens manufacturer + characteristic (Cooke, Panavision, Zeiss). Anamorphic vs spherical. Lens distortion behavior. Critical foundation.

Cinematography + lighting

Hiểu cinematography craft — DP work language. Lighting setup (3-point, naturalistic, stylized). Color temperature + meter. Hiểu DP intent — supervisor support not interfere DP vision.

VFX pipeline knowledge

Hiểu post pipeline: tracking (PFTrack, 3DEqualizer), compositing (Nuke), 3D (Maya, Houdini), lighting (Katana). Hiểu what post need — capture right data. Solve technical problem on set with post mindset.

Tool: HDRI capture, LiDAR, color chart

Field equipment: Spheron / Ricoh Theta Z1 (HDRI), Faro / Leica LiDAR scanner, Macbeth color chart, chrome + grey ball (for light reference), tape measure + laser distance measure, tablet for shot tracking (FileMaker, custom app).

Communication on set

Politics + diplomacy. On-set culture intense, DP + Director focus performance. Supervisor must integrate seamless — give info when needed, stay out of way otherwise. Clear concise communication. Soft skill critical.

Pressure handle + decision speed

Production day cost $100K-$500K+. Time precious. Decision speed matter. Problem solve under pressure. Calm + measured demeanor.

04Làm cách nào để trở thành On-set VFX Supervisor?

1
Học VFX / Cinematography 3–5 năm

Background: VFX / Film Production / Cinematography. VN: SKDA, FPT Arena, Đại học SKĐA Hà Nội. Quốc tế: Gnomon School VFX, SCAD Film + VFX, Bournemouth NCCA, AFI Cinematography. Plus hands-on shooting experience — short film + indie production. Hybrid background (VFX + cinematography) ideal.

2
VFX Artist 3–6 năm

Start as VFX Artist trong post — Compositor, Tracker, FX Artist. Build deep pipeline knowledge. VN VFX studio: Bad Clay Studio, Sparx*, Glassbox Animation, Tinker Studio. International: ILM, Weta, DNEG, Framestore, MPC.

3
Senior VFX Artist + on-set experience 5–10 năm

Senior post artist. Start visit set occasional — learn production language. Volunteer for small shoot to gain experience.

4
On-set VFX Coordinator / Data Wrangler 8–12 năm

Step into on-set role — junior level. Assist Supervisor. Build on-set craft + diplomatic skill.

5
On-set VFX Supervisor 12–18 năm

Lead on-set VFX. Major project responsibility. Build reputation through repeated success.

6
Sự nghiệp On-set VFX Supervisor

Long-term path: (1) VFX Supervisor (overall) — manage entire VFX, both production + post; (2) Visual Effects Producer — business + producing side; (3) Production company VFX Head; (4) 2nd Unit Director — direct VFX-heavy scene; (5) Director transition — apply visual sense; (6) Consultant + educator. Lương: VN On-set VFX Supervisor mid 70-150M/tháng (during production); senior 150M-300M+/tháng. International: $1,500-$3,000/day mid; $3,000-$6,000/day senior. Top supervisor (Marvel, Star Wars level) $7,000-$15,000/day plus credit + residual. Career long-tail — physically demanding (long shooting day, location travel) but creatively + financially rewarding.

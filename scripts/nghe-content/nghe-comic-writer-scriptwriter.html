Khi Stan Lee viết Spider-Man + X-Men + Fantastic Four trở thành Marvel founder, khi Alan Moore viết Watchmen + V for Vendetta + From Hell breakthrough comic literary, khi Neil Gaiman viết The Sandman thay đổi industry, khi Robert Kirkman viết The Walking Dead + Invincible truyền cảm hứng thế hệ — đó là tác phẩm của Writer / Scriptwriter comic.

Writer / Scriptwriter comic là chuyên gia viết script + plot cho comic / manga / webtoon — không vẽ, chỉ viết. Marvel / DC nhiều writer iconic (Stan Lee, Alan Moore, Neil Gaiman, Frank Miller, Brian K. Vaughan). Partner với Artist để hiện thực hóa câu chuyện. Critical role — story foundation everything. Without Writer, Artist guess.

01Writer / Scriptwriter comic là ai?

Writer / Scriptwriter comic là chuyên gia viết script cho sequential art. Phân loại theo format: (1) American comic Writer — Marvel, DC, Image, Dark Horse, Boom!; (2) Manga Writer / Story Editor — Japanese manga magazine; (3) Webtoon Writer — Korean platform; (4) Graphic novel Author — single book; (5) European bande dessinée Writer. Phân loại theo style: (1) Full script — detailed panel-by-panel description (Marvel modern style); (2) Plot-first / Marvel style classic — Writer plot, Artist draw, Writer dialogue last; (3) Loose collaboration — outline only. Phân loại theo career: (1) Aspiring / indie; (2) Published Writer; (3) Established Writer — known name; (4) A-list Writer — Alan Moore, Neil Gaiman tier. Phân loại theo employer: (1) Big 2 (Marvel + DC) staff; (2) Image Comics + creator-owned; (3) Webtoon platform; (4) Independent / self-publish; (5) Vietnam — Comicola, Phan Thị, Kim Đồng.

Khác với Comic Artist / Penciller — Writer text, Artist visual. Sometimes same person (Hayao Miyazaki manga, Frank Miller). Khác với Novelist — Novelist prose, Comic Writer visual + dialogue heavy. Khác với Screenwriter — Screenwriter for screen, Comic for print panel. Many overlap (Brian K. Vaughan write both). Khác với Editor — Editor refine, Writer create. Hợp tác chặt với: Artist (penciller, inker, colorist), Editor, Publisher, Letterer, Cover Artist, Movie / TV adaptation team (when IP adapted), Fanbase + reader.

02Công việc của một Writer / Scriptwriter comic

1. Story + character + world build

Foundation. High-concept hook: pitch one-line. Character create: protagonist + antagonist + supporting. World build: rule, geography, culture. Story arc: multi-issue, multi-volume planning. Series bible: reference doc cho team. Pitch to publisher / editor: cover art + first chapter spec.

  • Concept + character
  • World + arc
  • Series bible + pitch

2. Script write panel-by-panel — primary work

Core. Issue outline: page-by-page beat. Full script write: panel-by-panel — panel count per page, panel description (visual action), dialogue + caption + sound effect. Page count: 22 page standard for monthly American comic. Marvel style: panel description, then dialogue. Pacing planning: cliffhanger per issue. Multi-issue arc: 6-12 issue saga plan.

03Writer / Scriptwriter comic cần giỏi gì?

📖 Story structure mastery
👥 Character + voice
💬 Dialogue snappy
🎨 Visual thinking + panel sense
📚 Comic literacy deep
⏰ Discipline + monthly grind
Story structure mastery

Single issue arc + multi-issue arc + season arc. Beat structure. Page turn beat. Read: "Understanding Comics" (McCloud), "Story" (McKee), "Save the Cat!" (Snyder), "Words for Pictures" (Bendis).

Character + voice

Character voice distinct. Motivation. Arc. Backstory. Read: "Creating Characters" (Sheldon), study Alan Moore + Neil Gaiman + Brian K. Vaughan dialogue.

Dialogue snappy

Brief (panel space limit). Punchy. Character voice. Subtext. Less is more. Read: "The Anatomy of Story" (Truby), "The Art of Comic Book Writing" (Pak + Pak).

Visual thinking + panel sense

Thinking in panel + page + spread. Panel rhythm. Page turn beat. Read: "Making Comics" (McCloud), "Comics and Sequential Art" (Eisner).

Comic literacy deep

Read voraciously — Western superhero, indie, manga, webtoon, French. Movement history. Genre convention. Read: "Reading Comics" (Wolk), "Supergods" (Morrison).

Discipline + monthly grind

Monthly schedule brutal. Multiple title parallel for income. Deadline religion. Self-discipline. Solo work mostly.

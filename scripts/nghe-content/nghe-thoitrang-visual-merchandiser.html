01Visual Merchandiser là ai?

Visual Merchandiser (VM) là chuyên gia visual storytelling cho retail. Phân loại theo focus: (1) Window Display VM — window dressing, eye-catching street; (2) In-store VM — interior, table, mannequin, shelf; (3) Flagship VM — large concept store; (4) Visual Marketing — campaign + cross-channel; (5) Lookbook + Catalog VM — print + digital. Phân loại theo career: (1) VM Assistant; (2) VM; (3) Senior VM / Regional VM; (4) Head of VM / National VM; (5) Global VM Director. Phân loại theo employer: (1) Luxury brand — LVMH, Kering, Richemont, Dior, Hermès; (2) Fast fashion — Zara, H&M, Uniqlo, Mango; (3) Department store — Selfridges, Lotte, Diamond Plaza; (4) Tech retail — Apple Store, Samsung Experience; (5) Local brand — Vingroup retail, Routine, IVY moda, Catsa. Phân loại theo product: (1) Fashion apparel; (2) Accessories + jewelry; (3) Cosmetic + beauty; (4) Tech + electronics; (5) Home + lifestyle.

Khác với Store Designer / Retail Designer — Store Designer architecture + fixture (permanent), VM display + product (changeable). Khác với Stylist — Stylist client-side (celebrity, editorial), VM retail-side. Khác với Brand Designer — Brand brand-system, VM apply brand at retail. Khác với Buyer / Merchandiser commercial — Buyer what to sell, VM how to display. Hợp tác chặt với: Buyer + Merchandiser (commercial), Store Manager + Staff, Marketing team, Brand Designer, Photographer (lookbook), Stylist (lookbook), Construction + Fabricator.

03Visual Merchandiser cần giỏi gì?

👁️ Visual eye + composition
🎨 Color + styling sense
🏗️ 3D + spatial thinking
📊 Sales + customer literacy
💪 Physical + hands-on
📷 Photo + Photoshop + 3D viz
Visual eye + composition

Symmetry, balance, focal point. Repetition + variation. Visual rhythm. Read: "Visual Merchandising and Display" (Pegler + Kong), "Silent Selling" (Diamond).

Color + styling sense

Color theory. Seasonal palette. Styling trend. Outfit coordinate. Layering. Read: "Color: A Workshop for Artists and Designers" (Edwards), fashion magazine subscribe.

3D + spatial thinking

Layout in 3D space. Sight line. Customer journey. Traffic flow. SketchUp / Cinema 4D basic. Hand sketch.

Sales + customer literacy

Hiểu retail metric (sales per sqft, conversion, basket size). Customer journey + dwell time. Hot zone vs cold zone. Read: "Why We Buy" (Underhill).

Physical + hands-on

Climb ladder. Carry mannequin. Long hour standing. Power tool basic (drill, glue gun). Tape measure literate. Field work + creative work mix.

Photo + Photoshop + 3D viz

Photoshop + Illustrator mockup. SketchUp + Cinema 4D 3D viz cho concept present. iPhone photography for documentation + report.

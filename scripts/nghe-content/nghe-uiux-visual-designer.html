01Visual Designer là ai?

Visual Designer là chuyên gia visual storytelling cho product + brand. Phân loại theo focus: (1) Product Visual Designer — in-product illustration + icon + visual; (2) Brand Visual Designer — brand-side, identity + collateral; (3) Marketing Visual Designer — campaign, social, landing; (4) Editorial Visual Designer — content, magazine, blog; (5) Illustration-heavy Visual Designer. Phân loại theo career: (1) Junior Visual Designer; (2) Visual Designer; (3) Senior Visual Designer; (4) Lead / Principal Visual Designer; (5) Art Director / Design Director. Khác với Illustrator — Illustrator illustration-only, Visual Designer broader (illustration + layout + photo direction). Hợp tác chặt với: UI + UX Designer, Brand Designer, Illustrator, Photographer + Art Director, Marketing team, Product Manager, Motion Designer.

03Visual Designer cần giỏi gì?

🎨 Visual fundamentals
✏️ Illustration + drawing
💻 Figma + Adobe Suite + Procreate
📐 Typography + layout
📷 Photography direction
🌈 Color theory + brand
Visual fundamentals

Composition. Hierarchy. Balance. Contrast. White space. Read: "The Elements of Graphic Design" (White), "Universal Principles of Design" (Lidwell), "Design as Art" (Munari).

Illustration + drawing

Spot illustration. Hero image. Character. Editorial style. Read: "The Lost Art of Drawing the Head" (Massey), "Illustration School" (Kudo). Sketch daily.

Figma + Adobe Suite + Procreate

Figma primary. Adobe Illustrator + Photoshop. Procreate illustration. After Effects motion. Cinema 4D 3D illustration (trend).

Typography + layout

Type pairing. Hierarchy. Grid. Read: "Thinking with Type" (Lupton), "The Elements of Typographic Style" (Bringhurst).

Photography direction

Composition. Lighting. Color grading. Photographer brief. Style guide. Read: "Read This If You Want to Take Great Photographs" (Carroll).

Color theory + brand

Color harmony. Brand palette. Cultural color meaning. Read: "Interaction of Color" (Albers), "Designing Brand Identity" (Wheeler).

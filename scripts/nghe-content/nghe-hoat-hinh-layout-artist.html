Trong "Coco" của Pixar, khi camera bay qua land of the dead với hàng triệu petal cempasúchil + thành phố lung linh — đó là Layout Artist staged scene. Trong "Spirited Away" Ghibli, khi Chihiro băng cầu sang bath house, mỗi góc camera + position character đều được Layout Artist plan trước.

Layout Artist là cinematographer + production designer của animation — họ define camera (angle, lens, movement), staging (position character + prop trong scene), và composition. Là cầu nối critical giữa Storyboard và Animator.

01Layout Artist là ai?

Layout Artist là nghệ sĩ define camera + staging cho scene trong animation. Phân loại theo pipeline: (1) 3D Layout Artist — set up camera + character + prop trong 3D scene (Maya, Blender) — define shot composition + camera move; (2) 2D Layout Artist (BG Layout) — vẽ background scene + position cho 2D animation (Ghibli, classic Disney). Trong Pixar/DreamWorks pipeline có thêm Rough Layout (quick blocking, low-fi) → Final Layout (polished, animation-ready). Specialization theo content: action scene specialist, dialogue scene, environment establishing shot.

Khác với Storyboard Artist (2D sketch panel, narrative) — Layout dimensional, accurate. Khác với Animator (motion of character) — Layout sets up the stage, animator performs. Khác với Set Designer / Environment Artist (modeling environment) — Layout uses model, define composition. Khác với Director of Photography live action — Layout = virtual DP cho animation. Hợp tác chặt với: Director, Storyboard Supervisor, Animation Director, Environment Modeler, Animator, Editor.

  • Define camera angle + lens + movement
  • Stage character + prop trong scene
  • Translate storyboard → 3D space
  • Hand off scene cho animator

03Layout Artist cần giỏi gì?

📋 3D space + perspective
⚡ Timing + editing sense

Cinematography + camera language

Foundation. Hiểu film cinematography: lens choice (wide distort space, telephoto compress), shot size (extreme wide, wide, medium, close, extreme close), angle (low = power, high = vulnerable), camera movement (pan, tilt, dolly, crane, handheld). Study: "Cinematography: Theory and Practice" (Brown), "The Visual Story" (Bruce Block), watch film analysis ("Every Frame a Painting" YouTube).

Composition + visual storytelling

Master composition rule (and when to break): rule of third, leading line, frame within frame, negative space, depth layer. Each shot tells story qua composition — who is power, where focus, what emotion. Study painting (Vermeer, Caravaggio chiaroscuro), photography (Cartier-Bresson).

Tool: Maya, Blender, Unreal, Houdini

Maya — industry standard cho 3D animation feature (Pixar, DreamWorks, ILM). Blender — growing, indie + studio. Unreal Engine 5 — real-time virtual production (The Mandalorian, Loki). Houdini — procedural, technical layout. Storyboard Pro — for 2D layout. Cintiq + Wacom tablet.

3D space + perspective

Layout exist in 3D space. Strong perspective drawing foundation. Visualize 3D scene mentally. Comfortable manipulate camera + object in 3D space. Study: "Perspective Made Easy" (Norling), draw 3D thumbnail before scene build.

Timing + editing sense

Layout shot must fit edit timing. Hiểu pacing — when to hold shot, when to cut. Coordinate với editor + storyboard timing. Watch lot of film + identify editing pattern.

Direction interpretation

Receive note từ Director + Layout Supervisor. Interpret accurately. Translate verbal intent ("more intimate", "more menacing") into concrete camera choice. Iterate willingly. Strong communicator.

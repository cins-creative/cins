01Storyboard Artist hoạt hình là ai?

Storyboard Artist hoạt hình là chuyên gia animation pre-production storytelling. Phân loại theo career: (1) Storyboard Revisionist — entry, revise senior's board; (2) Storyboard Artist; (3) Senior Storyboard Artist; (4) Storyboard Lead / Head of Story — top role. Phân loại theo project: (1) Animation feature — Pixar, Disney, DreamWorks, Illumination; (2) TV animation series — Netflix series, Cartoon Network, Nickelodeon; (3) Animated commercial; (4) Game cutscene + cinematic; (5) Short film + indie. Phân loại theo style: (1) 3D animation board — most studio standard; (2) 2D animation board; (3) Stop-motion board — Laika, Aardman.

Khác với Film Storyboard Artist (live action) — Animation more iteration heavy + pitching culture. Khác với Layout Artist (3D layout from board) — Storyboard plan, Layout execute camera + composition 3D. Khác với Animator (move character) — Animation execute board. Khác với Visual Development Artist (look + style) — different. Hợp tác chặt với: Director, Head of Story, Other Storyboard Artist, Editor, Layout Artist, Animation Director, Production Designer.

02Công việc của một Storyboard Artist hoạt hình

1. Script + sequence assign + pre-thinking

Script read deep: per sequence, identify beat, mood, theme. Sequence assign: Head of Story assign sequence per Artist. Director meeting: align sequence's purpose, key beat. Reference: classic film, animation reference, similar comedy / drama sequence. Thumbnail pre-think: rough thumbnail per beat. Acting + comedy beat plan: where joke land, where emotion build.

  • Script + sequence
  • Director align
  • Reference + thumbnail

2. Storyboard draw — primary work

Core. Panel per beat: hundreds to thousand panel per sequence. Acting + expression + composition + camera. Comedy timing: pause, reaction, beat — board capture rhythm. Emotion build: facial expression, body language, framing. Camera + composition: angle, framing, movement. Animation feasibility: hiểu what Animator can execute, what too complex. Continuity: shot-to-shot logic, eye line, screen direction.

3. Pitch + Director note + revise

Pitch session: Storyboard Artist literally pitch sequence aloud — perform voice, sound effect, gesture. Pixar pioneer this culture. Director + Head of Story watch. Note take: copious feedback — "comedy weaker here", "emotion needs more", "camera too far", "redo this section". Revise: per note — sometimes minor (change angle), sometimes major (rework entire sequence). Multiple pitch round: 5-20+ pass per sequence on feature.

  • Pitch session
  • Director note
  • Revise multiple round

4. Animatic + handoff to layout / animation

Animatic build: storyboard + temp dialogue + temp music + sound effect → rough video — timing + flow check. Story reel: full feature animatic — Director watch entire film via storyboard. Iterate animatic: pace adjust, scene cut, add. Final approve: board lock. Handoff to Layout (3D feature): Layout Artist translate board to 3D shot. Handoff to Animator (2D): direct reference. Production support: revise occasionally during animation.

Drawing + acting

Capture acting + expression in pose. Body language. Force + weight. Read: "The Animator's Survival Kit" (Williams), "Drawn to Life" (Stanchfield), "Force: Dynamic Life Drawing" (Mattesi).

Pitching + performance

Pitch aloud với voice + gesture. Sell sequence to Director. Pixar culture pitching is half the job. Practice mirror. Watch Pixar artist pitch on extras.

Comedy + emotion timing

Comedy timing — pause, reveal, callback. Emotional beat. Subtext. Read: "The Comic Toolbox" (Vorhaus), "The Animator's Workbook" (White), watch Pixar + Studio Ghibli + Looney Tunes deeply.

Cinematography literacy

Camera language. Composition. Read: "The Filmmaker's Eye" (Mercado), "The Five C's of Cinematography" (Mascelli), "Directing the Story" (Glebas).

Storyboard Pro + iPad

Toon Boom Storyboard Pro industry standard animation. iPad + Procreate. Photoshop. Premiere for animatic. Maya / Blender basic for 3D ref.

Speed + iteration

Hundreds panel/sequence + multiple revision pass. Volume + speed critical. Comfortable với constant revise. Detached from any single drawing.

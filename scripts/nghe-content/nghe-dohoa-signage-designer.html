01Signage Designer là ai?

Signage Designer là chuyên gia design wayfinding + signage system. Phân loại theo project type: (1) Transportation — sân bay (Changi, Heathrow), train station (Tokyo Metro), bus terminal; (2) Healthcare — hospital (Vinmec, Mayo Clinic), clinic; (3) Education — university campus, school; (4) Cultural — museum (Louvre, MoMA), gallery, library; (5) Retail + commercial — mall (Vincom, AEON), retail district; (6) Civic — government building, public park; (7) Corporate campus — Apple Park, Google HQ. Phân loại theo signage type: (1) Directional — arrow + destination; (2) Identification — name room, building; (3) Informational — map, schedule; (4) Regulatory + safety — exit, warning.

Khác với Graphic Designer (print + digital) — Signage Designer 3D + environmental, must consider physical + spatial. Khác với Architect (building) — Signage Designer overlay on architecture. Khác với Environmental Designer (broader physical space) — Signage Designer narrower (sign + wayfinding focus). Khác với UX Designer (digital) — similar mindset (navigate user), different medium. Hợp tác chặt với: Architect, Interior Designer, Brand team, Operations + facility team, Fabricator + sign shop, Accessibility consultant.

02Công việc của một Signage Designer

1. Site analysis + user research

Site visit + walk-through: experience the space as user. Identify confusing area, decision point, pain. Stakeholder interview: với operations + facility + visitor / patient. Existing condition analysis: photo + measure existing signage (if renovation). User journey map: từ entry to destination. Decision point identify: where người need direction. Demographic + accessibility consider: language, visual impairment, mobility.

03Signage Designer cần giỏi gì?

🧭 Wayfinding system thinking
🔤 Typography + legibility
📐 CAD + drafting
🛠️ Material + fabrication
♿ Accessibility + code
🏛️ Architecture literacy
Wayfinding system thinking

System logic. Hierarchy. Naming convention. Decision tree mental model. Read: "Wayfinding: People, Signs, and Architecture" (Arthur + Passini), "Sign Language" (Schultz), "Wayfinding Handbook" (Calori).

Typography + legibility

Typography for distance + low light + glance. Sans-serif typical (DIN, Frutiger, Helvetica Neue, Roboto, IBM Plex). Contrast + spacing critical. Read: "The Elements of Typographic Style" (Bringhurst), "Sign Language" (Schultz).

CAD + drafting

AutoCAD standard. Vectorworks. Rhino + SketchUp for 3D study. Illustrator for sign artwork. InDesign for sign schedule + manual. Layered drawing for fabrication.

Material + fabrication

Aluminum, stainless steel, acrylic, MDF, vinyl, dibond, illuminated (LED, neon). Mount method (post, wall, ceiling, free-standing). Visit fabricator workshop. Hands-on knowledge.

Accessibility + code

ADA (Mỹ), Equality Act (UK), Việt Nam QCVN regulation. Braille + tactile required certain sign. Height + clearance regulation. Color contrast ratio. WCAG-inspired physical signage standard.

Architecture literacy

Read architectural drawing. Hiểu space + spatial flow. Coordinate với architect + interior. Read: "Architecture: Form, Space, and Order" (Ching).

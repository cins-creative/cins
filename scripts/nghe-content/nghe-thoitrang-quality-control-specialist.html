Bạn cầm áo Nike và cảm nhận seam straight, stitch even, fabric clean. Bạn nhận đơn Shein và phát hiện loose thread + uneven dyeing → return. Sự khác biệt giữa quality stable brand + flaky brand = Quality Control Specialist.

Quality Control Specialist (QC) thời trang là chuyên gia kiểm tra chất lượng garment ở mọi giai đoạn production — fabric inspection, mid-production check, final inspection trước ship. Đảm bảo product brand standard + customer expectation. Critical role bridge giữa factory output + brand reputation. One bad batch = customer return + brand damage.

02Công việc của một Quality Control Specialist thời trang

1. Fabric + raw material inspection

First gate. Receive fabric + trim shipment from supplier. 4-point inspection: per yard inspect for defect (hole, stain, weave fault). Shade matching: lot-to-lot consistency với approved standard (Datacolor spectrophotometer). GSM check: weight per square meter accurate. Hand feel: subjective quality. Test report: shrinkage, pilling, color fastness (third-party lab). Accept / reject entire batch.

  • Shade + GSM
  • Lab test

2. Pre-production + mid-production check

Pre-production sample (PPS): first off-line sample inspect against spec. Approve hoặc reject before mass production start. Mid-production check (DUPRO — During Production): when 30-50% production complete, sample inspect for trend. Catch problem early. Measurement check: per tech pack tolerance (chest, length, sleeve, etc.). Construction check: seam, stitch density, finishing. Defect classification: major, minor, critical.

3. Final inspection + AQL

Critical gate. Final inspection at factory before ship. AQL sampling: per AQL 2.5 (common for apparel) — pull random sample size based on lot size. Inspect each sample: visual (defect), measurement, function (zipper, button, snap). Classify defect: major, minor, critical (defect that affect safety / usability — auto-reject). Accept / reject lot: based on # defect vs AQL limit. Re-inspection if rejected — factory rework + resubmit.

4. Report + improve + audit

Inspection report: detail finding, photo, defect rate, decision (Accept / Reject / Hold). Send to Production Manager + brand. Vendor performance track: per-factory quality score historic. Root cause analysis: when defect repeat, identify cause (machine, operator, design). Recommend improvement. Factory audit: periodic comprehensive review of factory capability + system. Compliance audit: BSCI, SA8000, customer requirement.

Technical garment knowledge

Hiểu garment construction. Seam type. Stitch type. Fabric type + behavior. Trim. Finishing. Read: "Quality Assurance for Textiles and Apparel" (Kadolph), ASTM + AATCC test standard.

Measurement + spec literacy

Read tech pack. Measure accurately. Tolerance understanding. Spec sheet interpret. Tool: measurement tape, caliper, digital scale, light box, magnifier.

AQL + statistical

AQL (Acceptable Quality Level) standard ISO 2859. Sampling plan. Statistical process control basic. Defect rate calculate. Pareto analysis cho root cause.

Document + report

Detail report writing. Photo documentation. Excel + reporting software. Concise + objective tone. Read: write clear technical document.

Vendor relationship

Diplomatic when reject. Constructive when feedback. Build factory relationship — they trust QC fair, not adversarial. Multi-cultural (international factory).

Attention to detail

Spot small defect. Pattern recognition. Patience + thoroughness. Long hour focus capacity.

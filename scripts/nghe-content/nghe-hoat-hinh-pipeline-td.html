Pipeline TD (Technical Director) hoạt hình là chuyên gia build + maintain pipeline production của animation studio. Tool, script, automation, asset management — tất cả invisible glue connecting department. Role bridge giữa technical engineering + creative production. Critical cho studio efficiency — millions $ savings qua tốt pipeline.

01Pipeline TD hoạt hình là ai?

Pipeline TD là chuyên gia phát triển software + tool + workflow tự động cho animation studio. Phân loại theo focus: (1) Asset Pipeline TD — asset management system, version control, dependency tracking; (2) Department TD — specific department tool (Rigging TD, Lighting TD, Animation TD); (3) Production Pipeline TD — track production status, deadline, artist workload; (4) Render Pipeline TD — render farm management, job submission, optimization; (5) USD Pipeline TD — Pixar USD framework integration (modern standard); (6) Show Pipeline TD — per-show customization. Phân loại theo company: Major studio (Pixar, Disney, DreamWorks, Sun Wolf) — dedicated pipeline team; Mid studio — pipeline TD wear multiple hat; Indie — generalist TD.

Khác với Software Engineer general — Pipeline TD domain-specific (CGI animation pipeline). Khác với Technical Animator / Rigger — Pipeline TD tool / infrastructure; TA / Rigger artist work. Khác với IT — Pipeline TD CG-tool-specific; IT general infrastructure. Khác với R&D — Pipeline TD production-applied; R&D long-term research. Hợp tác chặt với: Pipeline Supervisor / CTO, Department Supervisor, Producer, Artist all department, IT team, R&D team, External vendor.

  • Develop pipeline tool + script
  • Maintain asset management system
  • Solve technical bottleneck
  • Support artist production daily

03Pipeline TD hoạt hình cần giỏi gì?

Python + C++ coding

Python — primary language, all DCC tool support Python API. Master OOP, decorator, multithreading, async, packaging. C++ for performance — Maya plugin, Houdini SOP, Nuke plugin. JavaScript / React for web tool. Course: Python for Maya, HoudiniSchool, RebelWay.

DCC tool API (Maya, Houdini, Nuke)

Pipeline TD bridge tool. Master API: Maya Python + MEL + Maya API C++, Houdini HOM (Python) + VEX, Nuke Python + NDK, Blender bpy. Hiểu DCC architecture — node graph, dependency, evaluation.

Animation pipeline knowledge

Hiểu pipeline động: pre-production → production → post. Each department workflow: modeling, rigging, animation, lighting, compositing. Without artist understanding, tool not fit. Best Pipeline TD often ex-artist.

USD framework

Pixar USD = modern animation pipeline standard. Layered composition, hierarchical scene description, asset interop. Adopted by major studio. Read: Pixar USD documentation, OpenUSD.org.

System architecture + database

Build production system. Database (SQL, MongoDB, ShotGrid Postgres). REST API. Caching. Distributed system. Cloud (AWS, Azure). Storage architecture (NAS, object storage).

Artist empathy + communication

Tool serve artist. Empathy listen pain point. Translate technical → simple language. Patience explain bug + workaround. Soft skill critical for adoption.

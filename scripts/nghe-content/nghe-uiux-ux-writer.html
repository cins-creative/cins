03UX Writer cần giỏi gì?

✍️ Writing concise + clear
🎙️ Voice + tone craft
🎨 Design literacy + Figma
🧠 UX + user mental model
🌐 Localization awareness
📊 Data + A/B test literacy
Writing concise + clear

Plain language. Active voice. Action verb. Short sentence. Read: "On Writing Well" (Zinsser), "Strategic Writing for UX" (Podmajersky), "Microcopy" (Yifrah).

Voice + tone craft

Brand personality define. Tone adapt per context (success vs error vs onboarding). Voice consistent + tone flexible. Read: "Nicely Said" (Kissane + Bowles), "Conversational Design" (Hoober).

Design literacy + Figma

Work directly in Figma. Hiểu design constraint. Spatial limit. Component reuse. Auto layout. Variable.

UX + user mental model

Hiểu user task + flow. Write per user not per business. Avoid jargon. Read: "Don't Make Me Think" (Krug), "100 Things Every Designer Needs to Know About People" (Weinschenk).

Localization awareness

Avoid idiom. Avoid culture-specific reference. Variable + placeholder syntax (i18n). String expansion (German 30% longer than English). Read: "Found in Translation" (Kelly).

Data + A/B test literacy

A/B test variant design. Statistical significance basic. Click-through rate, conversion. Read: "Lean Analytics" (Croll + Yoskovitz).

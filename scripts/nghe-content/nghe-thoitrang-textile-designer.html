Khi áo dài có lụa Vạn Phúc dệt tinh xảo, khi Marimekko's Unikko poppy pattern thành Finnish icon 60+ năm, khi Burberry's check pattern recognizable globally — đó là tác phẩm của Textile Designer.

Textile Designer là chuyên gia thiết kế vải — print, weave, knit, embroidery. Critical bridge giữa fabric mill + fashion designer. Specialty: print designer, weave designer, knit designer. Career path tại textile mill + brand in-house + freelance. Without Textile Designer, fashion designer chỉ có generic fabric, no signature material identity.

01Textile Designer là ai?

Textile Designer là chuyên gia design fabric + textile. Phân loại theo specialty: (1) Print designer — surface print pattern (largest); (2) Weave designer — woven fabric structure (twill, satin, jacquard); (3) Knit designer — knitted fabric (jersey, rib, intarsia); (4) Embroidery designer — surface embellishment; (5) Smart textile designer — emerging (conductive thread, e-textile). Phân loại theo application: (1) Fashion apparel; (2) Home textile — bedding, curtain, upholstery; (3) Technical textile — automotive seat, medical, military; (4) Luxury fabric — silk, wool, cashmere mill. Phân loại theo employer: (1) Textile mill — Vạn Phúc, Sông Hồng Textile, Phong Phú; (2) Fashion brand in-house; (3) Design studio; (4) Freelance.

Khác với Fashion Designer (apparel design) — Textile fabric design, Fashion use textile. Khác với Surface Pattern Designer (print-only application broader) — Textile narrower fashion + technical specific. Khác với Material Engineer (engineering + chemistry) — Textile design + aesthetic + craft. Khác với Costume Designer — different context. Hợp tác chặt với: Fashion Designer, Textile Mill engineer + production, Color Stylist, Material sourcer, Trend Forecaster (WGSN), Buyer, Sustainability consultant.

03Textile Designer cần giỏi gì?

🧶 Fabric + fiber knowledge
🎨 Pattern + color design
💻 NedGraphics + Illustrator
🏭 Mill + production process
🌱 Sustainability literacy
📊 Trend + commercial sense
Fabric + fiber knowledge

Fiber (cotton, wool, silk, polyester, viscose, modal). Yarn (count, ply, twist). Weave structure (plain, twill, satin, jacquard). Knit (jersey, rib, interlock). Read: "Textiles: A History" (Schoeser), "Textile Designer's Manual" (Bowles + Isaac).

Pattern + color design

Print pattern repeat. Color theory + palette. Pantone reference. Read: "Pattern Design" (Wilson), "Color and Design for Textiles" (Mathur).

NedGraphics + Illustrator

Adobe Illustrator + Photoshop basic. NedGraphics, Pointcarré, EasyWeave, ArahWeave textile specialty. Wilcom embroidery. Shima Seiki / Stoll Compact (Knit).

Mill + production process

Spinning, weaving, knitting, dyeing, finishing. Visit mill regularly. Hiểu loom + dye process. Reactive vs disperse dye. Finishing (mercerizing, calendering). Read: "Textile Technology" (Hatch).

Sustainability literacy

GOTS organic certification. Recycled fiber. Waterless dye. Circular economy. EU regulation (REACH, PEF). Read: "Sustainable Fashion" (Fletcher).

Trend + commercial sense

WGSN, Pantone, Première Vision report. Hiểu what sell, what fail. Brand alignment. Margin + cost awareness.

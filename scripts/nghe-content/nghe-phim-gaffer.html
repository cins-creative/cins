Khi xem Blade Runner 2049 với neon glow đặc trưng Roger Deakins, khi 1917 long take được lighting xuyên suốt phức tạp, khi VN phim Ròm ánh sáng ngõ hẻm gritty — tất cả đều có DP định vision lighting. Nhưng người execute mỗi đèn, mỗi điện, mỗi rigging — là Gaffer.

Gaffer = Chief Lighting Technician. Nếu DP là "kiến trúc sư ánh sáng", Gaffer là "chỉ huy công trường thi công ánh sáng" — execute vision với toàn bộ team electrician + truck đèn + cable + power source.

01Gaffer là ai?

Gaffer (Chief Lighting Technician) là trưởng bộ phận ánh sáng technical trên set phim. Tên đến từ "gaffer" — instrument dài có hook (cây sào gậy) dùng để adjust đèn cao thời kỳ đầu. Phân loại theo project scale: (1) Feature Gaffer — Hollywood blockbuster, team 8–30 electrician; (2) Indie / TV Gaffer — smaller crew 2–5 electrician; (3) Commercial Gaffer — fast-paced campaign shoot; (4) Music Video Gaffer — creative + ngắn schedule.

Khác với Director of Photography (DP/Cinematographer) (creative vision: framing, lighting style, mood) — Gaffer execute lighting. Khác với Key Grip (camera support + lighting modifier non-electric: flag, frame, silk) — Gaffer chuyên electrical. Khác với Best Boy Electric (Gaffer's #2 — gear logistics, crew schedule) — Gaffer là trưởng. Hợp tác chặt với: DP, Key Grip, Best Boy Electric, Electricians, Camera Operator, Production Designer, 1st AD.

  • Execute lighting plan của DP
  • Manage electrical crew + truck
  • Rig + safety check fixture, cable, power
  • Solve problem real-time on set

Vị trí trong quy trình sản xuất

Gaffer join project trong pre-production phase (2–4 tuần trước shoot). Báo cáo cho DP; tổ chức + trả lương dưới Production Manager / Line Producer. Lead team: Best Boy Electric (#2), Electricians 2–20, Rigging Gaffer + crew (large project). Phối hợp với: Key Grip (camera + non-electric lighting modifier), Camera Crew, 1st AD. Workflow daily: pre-light morning (1–3 hours setup) → shoot day → wrap night (1 hour). Feature film shoot 40–80 days; commercial 1–5 days; TV episode 7–10 days. Gear: lighting truck (3-ton hoặc 5-ton), HMI, LED panel, tungsten, cable, generator, distro box.

02Công việc của một Gaffer

1. Pre-production — lighting plan + gear list

Sit với DP early: discuss script + location + lighting style. Read script — note hint scene (night, sunset, romantic, noir). Visit location — measure power capacity, identify hang point, sun direction. Build lighting plan per scene (diagram: where fixture, what type, intensity). Compile gear list / equipment list: HMI count, LED count, cable run length, generator size, accessories. Order from rental house. Travel logistics.

  • Discuss với DP
  • Location recce
  • Gear list + rental

2. Set rigging + power management

On location: rig fixture — overhead hang (truss, rigging), stand-based, practical (lamp in scene). Power management — calculate amp load, distribute via distro box, ground properly. Cable run safe (cover with cable mat where crew walks). Generator placement (sound-quiet location). Safety always first — electrical fault = fire/electrocution risk.

  • Rig fixture overhead + stand
  • Power distribution
  • Safety protocol

3. Shoot day execution + DP collaboration

On set: DP call lighting note ("more soft on her face", "kick on background", "warm side fill"). Gaffer translate → instruct electrician execute. Energy intense — 12–14 hour shoot day.

03Gaffer cần giỏi gì?

⚡ Electrical knowledge + safety
🛠️ Lighting fixture mastery
✨ Hiểu lighting aesthetic + cinematography
💬 Lead crew + communication
📋 Planning + logistics
🎨 Problem-solve + improv
Electrical knowledge + safety

Foundation — Gaffer manage electricity. Hiểu deep: amp = watt / voltage, 3-phase power, generator capacity, cable gauge (10AWG, 4/0, banded loops), grounding, GFCI. Certification preferred: OSHA 30, IATSE Local 728 union training (US). VN: training tại VFA, ĐH Sân khấu Điện ảnh.

Safety = non-negotiable. Electrocution + fire = career-ending + criminal.

Lighting fixture mastery

Hundred fixture: HMI (ARRI M-series, M40, M90); Tungsten (ARRI 1K, 2K Junior, 5K Senior, 10K Tener); LED (ARRI SkyPanel S30/S60, Litepanels, Aputure Nova/600x, Kino Flo). Each has different output, color temp, modifier compatibility. Master plug-in spec (each fixture amp draw critical).

Hiểu lighting aesthetic + cinematography

Gaffer ≠ pure technician — phải hiểu DP intent. Study cinematography: read "Cinematography: Theory and Practice" (Brown), "Painting with Light" (John Alton); watch movie critically (rewatch Roger Deakins, Emmanuel Lubezki, Gordon Willis work). Visit movie set, observe DP-Gaffer dynamic.

Lead crew + communication

Lead team 5–30 electrician. Decisive command. Clear instruction (precise terminology). Respect crew labor — Gaffer set tone (positive culture = retention). Negotiate gracefully với DP push-back. Manage union politics nếu union project.

Planning + logistics

Gaffer = mini production manager cho lighting dept. Excel/Google Sheets: gear list, crew schedule, daily report. Software: StudioBinder, Movie Magic Scheduling. Spreadsheet tracking amp load, rental cost, hours.

Problem-solve + improv

Set unpredictable. Sun cloud over → swap to HMI replace. Generator fail → backup. Power source short → re-route. Calm execution under pressure. Senior Gaffer have arsenal of "tricks of the trade" (China ball lantern + dimmer = soft fill in tight space).

04Làm cách nào để trở thành Gaffer?

1
Học Film / Electrical + on-job training

Background phù hợp: Film Production, Cinematography, Electrical Engineering. VN: ĐH Sân khấu Điện ảnh (Quay phim / Đạo diễn hình ảnh), VFA Vietnam Film Academy. Quốc tế: AFI Conservatory, USC SCA, NYU Tisch; technical school for electrical certificate. Many Gaffer self-taught + apprentice route.

2
Entry: 3rd / 2nd / "Set Lighting Technician"

Most-traveled path: Set Lighting Technician / Electrician. Tasks: coil cable, hold reflector, swap bulb, load truck. Low-prestige, physical labor — but learn fundamental. 1–3 năm. VN: through BHD, Galaxy, Mega GS commercial production, indie short films, MV.

3
Best Boy Electric

3–6 năm — promote Best Boy Electric (Gaffer's right hand). Manage crew schedule, gear logistics, truck. Learn Gaffer perspective inside. Build relationship với DPs.

4
Gaffer indie / commercial

6–10 năm — first Gaffer role. Start indie film, MV, commercial. Build reel + relationship với DP rising. Trust-based work — DP loyal Gaffer for years.

5
Gaffer feature + TV series

10–15 năm — Gaffer feature film, TV series. VN: work với top DP local (Nguyễn Vinh Phúc, Lý Thái Dũng, Trần Phong); international DP shoot VN (Roger Deakins shoot some scene VN, foreign productions). Day rate $500-$2K VN; international $1K-$3K USD/day.

6
Sự nghiệp Gaffer

Long-term path: (1) Top Hollywood/international Gaffer — partner consistent DP (Roger Deakins's Gaffer is Mark Roberts decades); (2) DP transition — Gaffer step up DP (rare but possible — Robert Yeoman started as Gaffer); (3) Production Designer crossover via lighting expertise; (4) Rental house ownership / consultant; (5) Educator. Lương: VN Gaffer indie 25–60M/tháng (project-based); Gaffer feature 60M-150M+/project; international top Gaffer $300-$500K+/năm. Pinnacle role technical leadership trên set.

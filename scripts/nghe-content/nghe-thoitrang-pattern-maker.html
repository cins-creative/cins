Một sketch tinh tế của Chanel, một sequin gown phức tạp của Elie Saab, một puffer jacket sportwear của Adidas — tất cả đều bắt đầu từ một file: tập rập 2D. Người tạo ra tập rập đó là Pattern Maker.

Pattern Maker là chuyên gia chuyển sketch fashion designer thành rập 2D — bản vẽ kỹ thuật chính xác mỗi miếng vải mà tailor + may xưởng dùng để cắt + sew. Là cầu nối critical giữa design vision (sketch) + manufacturing reality (garment). Foundation craft của fashion industry — không có Pattern Maker tốt, không có garment đẹp.

01Pattern Maker là ai?

Pattern Maker (cũng gọi Patternmaker, tiếng Việt: Người làm rập, Thợ rập) là chuyên gia tạo + grade rập (pattern) cho garment. Phân loại theo phương pháp: (1) Flat Pattern Maker — vẽ rập 2D từ block + measurement; (2) Draping Pattern Maker — drape muslin trên dress form, tạo rập 3D-first; (3) CAD Pattern Maker — digital pattern trong Gerber / Lectra / CLO3D. Phân loại theo category: Womenswear, Menswear, Childrenswear, Activewear / sportswear, Lingerie + intimate, Bridal + couture, Outerwear (jacket, coat). Phân loại theo company: Haute couture house (Chanel, Dior, Givenchy), Ready-to-wear brand (RTW), Mass market (H&M, Zara), Sportswear giant (Nike, Adidas), Independent designer, Factory in-house.

Khác với Fashion Designer (sketch + vision) — Pattern Maker translate sketch → technical reality. Khác với Sample Maker (sew first sample) — Pattern Maker create blueprint Sample Maker work from. Khác với Tailor (final garment for client) — Pattern Maker industrial production blueprint. Khác với Garment Technologist (manufacturing process + QC) — Pattern Maker pattern-specific. Hợp tác chặt với: Fashion Designer, Sample Maker, Grader (size grading), Garment Technologist, Production manager, Factory.

  • Translate sketch thành rập 2D chính xác
  • Grade rập cho multiple size
  • Test fit trên dress form / model
  • Optimize rập cho material efficiency

Pattern Maker — cầu nối design + manufacturing

Vị trí trong quy trình sản xuất

Pattern Maker work tại fashion house / brand / factory in-house / independent freelance. Báo cáo Production manager / Head of Pattern / Design Director. Workflow per garment: receive sketch + spec từ Designer → study + research method → create first pattern (1-3 ngày) → fit test on dress form + model (1-2 vòng) → refine + finalize → grade for size range (1-2 ngày). Total: 1-2 tuần per style. Collection 30-50 style per season.

02Công việc của một Pattern Maker

1. Receive sketch + analyze garment

Receive technical sketch + spec sheet (measurement, material, construction note) từ Fashion Designer. Analyze garment structure: silhouette, dart, seam, panel split, ease, drape character. Identify construction challenge: complex curve, asymmetric, layered. Tham khảo fabric swatch — different fabric drape khác → pattern adjust accordingly.

  • Sketch + spec read
  • Structure analyze
  • Material consider

2. Create first pattern (flat / drape / CAD)

Three method: Flat pattern — start from base block + manipulate (dart, slash, spread) per sketch — fastest, common. Draping — drape muslin trực tiếp trên dress form, mark + transfer to flat — couture / complex form. CAD — digital pattern trong Gerber AccuMark, Lectra Modaris, Optitex, CLO3D — modern factory standard. Many Pattern Maker combine all three.

  • CAD digital

3. Fit test + refinement

Cắt + may first sample (toile / muslin / cotton). Fit test on dress form hoặc fit model (live person sample size). Sit với Designer + Sample Maker. Identify issue: tight at shoulder, drag line at hip, not enough ease, dart placement wrong. Refine pattern. Iterate 2-3 round. Achieve perfect fit + drape per Designer vision.

4. Grading + production-ready

Finalized pattern in size sample (e.g., M). Grading: scale pattern for full size range (XS-XXL) — not uniform scaling, follow body proportion change. Tool: Gerber AccuMark + Lectra Modaris for industrial grading. Marker: layout pattern piece on fabric width optimal (minimize waste). Output: production-ready pattern + marker for factory. Documentation: spec sheet, construction note.

03Pattern Maker cần giỏi gì?

Pattern theory + body anatomy

Foundation. Hiểu human body proportion + measurement. Pattern theory: dart + slash + spread + true. Ease (positive ease, negative ease, design ease). Block + sloper system. Read: "Patternmaking for Fashion Design" (Helen Joseph-Armstrong), "Pattern Magic" (Tomoko Nakamichi) — Japanese cult classic.

Sewing + construction skill

Pattern Maker must understand sewing — what's possible, what causes problem in construction. Many Pattern Maker sew first sample themselves. Construction knowledge: seam type, finishing, interfacing, lining, closure.

Tool: Gerber, Lectra, CLO3D

Modern factory: Gerber AccuMark + Lectra Modaris standard. Optitex alternative. CLO3D / Browzwear — 3D garment simulation + pattern drape virtual. Adobe Illustrator — flat sketch + tech pack drawing. CAD essential for modern career.

Fabric knowledge + drape sense

Fabric drape behavior: cotton stiff vs silk flow vs jersey stretch vs denim bulk. Pattern must adjust for fabric. Hands-on fabric experience essential. Visit fabric show — Premiere Vision, Texworld.

Detail obsession + accuracy

Pattern critical to millimeter. 1mm error multiply across thousand garment = visible quality issue. Meticulous + accurate. Double check measurement.

Designer communication

Translate designer creative vision → technical reality. Sometime designer ask impossible — Pattern Maker explain constraint diplomatically + propose alternative. Soft skill important.

04Làm cách nào để trở thành Pattern Maker?

1
Học Pattern Making / Fashion Design 2–4 năm

Background: Pattern Making / Fashion Design / Garment Technology / Fashion Construction. VN: ĐH Mỹ thuật Công nghiệp (Khoa Thiết kế Thời trang), HCMUTE (Khoa Công nghệ May), ĐH Sư phạm Kỹ thuật TP.HCM, RMIT Fashion, Arena Multimedia Fashion. Quốc tế: FIT NYC (Patternmaking program — gold standard), Central Saint Martins, Parsons, Bunka Fashion College Tokyo, Royal College of Art London, ESMOD Paris.

2
Apprentice Pattern Maker 2–4 năm

Apprentice at established Pattern Maker or in-house at factory / brand. VN: Việt Tiến, Nhà Bè, Việt Thắng, May 10, TNG, Viet Tien (factory) hoặc independent designer brand. International: Atelier Chanel, Dior, Givenchy — apprentice opportunity rare but legendary. Learn from senior Pattern Maker.

3
Junior Pattern Maker 2–4 năm

Independent work. Pattern simple-medium complexity. Build speed + accuracy.

4
Mid Pattern Maker 4–8 năm

Complex pattern. Lead pattern team. Specialize: tailored, draped, knit, technical (sportswear, swimwear).

5
Senior Pattern Maker / Head of Pattern 8–15 năm

Senior craft. Lead entire pattern department. Mentor junior. Couture-level work cho luxury brand.

6
Sự nghiệp Pattern Maker

Long-term path: (1) Head of Pattern / Production Director; (2) Master Patternmaker at couture house — legendary status; (3) Independent freelance specialist — premium rate cho designer brand; (4) Educator — FIT, Bunka, university; (5) Garment Technologist transition — broader production role; (6) Founder + own studio. Lương: VN Pattern Maker junior 8-18M/tháng; mid 18-40M; senior 40-90M; Head of Pattern 90-180M+. International: $50K-$80K mid; $80K-$140K senior; Master $140K-$250K+. Pattern Maker craft eternally valuable — even with 3D CAD tool, human pattern judgment irreplaceable. Stable + respected career.

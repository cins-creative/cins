Set Designer phim là chuyên gia thiết kế set (bối cảnh) phim — từ technical drawing đến scale model. Báo cáo Production Designer + Art Director. Là technical-creative role trong Art Department. Translate vision into buildable plan cho carpenter + scenic painter.

01Set Designer phim là ai?

Set Designer phim là technical drafter / architect trong Art Department. Phân loại theo career: (1) Junior Set Designer / Draftsperson — entry; (2) Set Designer — full role; (3) Senior Set Designer; (4) Lead Set Designer. Phân loại theo project type: (1) Major film studio — Hollywood blockbuster, period epic; (2) TV series — Netflix series với big art department; (3) Theme park + immersive; (4) Commercial + music video. Phân loại theo specialty: (1) Period set — historical accuracy; (2) Fantasy / sci-fi set; (3) Contemporary set; (4) Practical effect set — set destruct cho action.

Khác với Production Designer (head, vision) — Set Designer execute Production Designer's design. Khác với Art Director (supervise Set Designer + production crew) — Set Designer report Art Director. Khác với Architect — film set built fast + last weeks (not year). Khác với Theater Set Designer (live theater) — film set built once for camera. Hợp tác chặt với: Production Designer, Art Director, Construction Coordinator, Construction Foreman, Scenic Painter, Set Decorator, Cinematographer (sight line + camera angle).

  • Scale model + render
  • Production Designer's vision execute

02Công việc của một Set Designer phim

1. Receive design + sketch + reference

Production Designer brief — concept sketch, mood board, reference image, look book. Discussion meeting: align on vision, dimension, key feature. Site survey: if set built at location (vs studio stage), survey site. Stage measurement: hiểu stage size constrain set design. Camera coverage discuss: với Cinematographer, hiểu camera angle + movement plan affect set layout.

  • Brief receive
  • Site + stage survey
  • Camera coverage

2. Technical drawing — primary deliverable

Core. Plan view: top-down. Elevation: front, side, back view. Section view: cut through set. Detail drawing: per element — door, window, molding, prop integration. Specification: material, finish, paint color. Hardware schedule: every door knob, hinge, lock. AutoCAD / VectorWorks / Rhino — industry standard tool. Drawing serve as build doc cho Construction team.

03Set Designer phim cần giỏi gì?

AutoCAD + VectorWorks

AutoCAD industry standard. VectorWorks Spotlight (theater + film). Rhino for complex form. SketchUp for quick 3D. Master 2D drafting + 3D model.

Architecture literacy

Architecture background helpful. Hiểu structure, building system. Read: "Architectural Graphics" (Ching), "Architecture: Form, Space, and Order" (Ching).

Film + camera literacy

Hiểu cinematography. Camera angle + lens + movement affect set design. Read: "The Filmmaker's Eye" (Mercado), "Production Design" (Sennett).

3D + render tool

V-Ray / Lumion for render. SketchUp Pro for quick 3D. Unreal Engine increasingly relevant (VR walkthrough). Photoshop for matte + treatment.

Material + construction

Hiểu material — wood, MDF, foam, fabric, paint, fiberglass. Construction method. Set building specific. Read: "Designing a Production" (Carroll).

Period + style research

Research period architecture + furniture + decor. Reference library. Visit museum + historic building. Hiểu architectural style: Renaissance, Victorian, Art Deco, Modernist, etc.

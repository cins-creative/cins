Space Planner là chuyên gia layout + workplace strategy. 02Công việc của một Space Planner

1. Programming + requirement gather

Stakeholder interview: với leadership + department head + HR — hiểu business need, headcount projection, work style. Survey + observation: existing space utilization data, peak occupancy. Program develop: spreadsheet — every space need (workstation, meeting room, focus room, social, support). Headcount + growth: 3-5 năm projection. Work style assess: assigned vs hot desk vs hybrid + percentage.

2. Test fit + option compare

Core deliverable. Test fit: quick layout option in CAD — fit program into building floor plate. Multiple option: 2-3 layout option for client compare. Block + stack: high-level zoning (department by floor or zone). Adjacency analysis: who need sit near who (Sales + Marketing, Engineering + QA). Circulation + flow: how people move through space, escape route. Quantity + efficiency check: utilization ratio, sqm per person.

After client choose direction. Detailed plan: every workstation, meeting room, room location, dimension. AutoCAD + Revit

AutoCAD standard for space planning. Revit BIM increasingly standard. SketchUp for 3D quick. BlocksCAD / Furniture Library. Cobra / Bluebeam markup. Read: "Mastering AutoCAD" (Omura).

Programming + spreadsheet

Excel master for program — pivot table, formula, sqm calculation. Headcount projection. Utilization analysis. Visualization (Tableau, Power BI). Workplace data analysis.

Workplace strategy

Hiểu workplace trend — activity-based working, hot desking, hybrid work, collaboration zone. Read: "Cool Spaces" (Lewis), "The Activity-Based Workplace" (Cushman). Follow Steelcase, Herman Miller research.

Furniture system literacy

Major manufacturer: Steelcase, Herman Miller, Haworth, Knoll, Kokuyo, Vitra. Workstation system (Answer, Resolve, Compose). Task chair, conference table, sofa. Specification + lead time + price knowledge.

Code + regulation

Building code (occupancy, egress, fire). Accessibility (ADA, QCVN). Worksafe regulation. Energy code. Permit process.

Client + stakeholder

Workshop facilitate (leadership session, focus group). Present option. Manage conflicting opinion. Diplomatic communication. Manage budget conversation.

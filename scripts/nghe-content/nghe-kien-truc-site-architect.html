Khi tòa Landmark 81 build đúng design intent + structural integrity + finish quality, khi Vincom Mega Mall đi từ paper drawing đến opening day smooth, khi villa hạng sang Da Lat hoàn thiện với material spec chính xác — đó là tác phẩm của Site Architect.

Site Architect là kiến trúc sư on-site giám sát construction — đảm bảo build đúng design intent + spec + code. Là bridge giữa design office + construction crew. Critical cho project quality + delivery timeline + budget control. Without Site Architect, construction drift from design + quality drop.

01Site Architect là ai?

Site Architect (Resident Architect / Architect-in-Charge) là kiến trúc sư representative on-site during construction. Phân loại theo project type: (1) Residential — villa, apartment, townhouse; (2) Commercial — office tower, retail, mixed-use; (3) Hospitality — hotel, resort; (4) Healthcare — hospital; (5) Education — school, university campus; (6) Infrastructure — airport, station, bridge. Phân loại theo employer: (1) Architecture firm — representative at site; (2) Owner / developer in-house — owner's rep; (3) Construction manager firm — design oversight.

Khác với Project Architect (lead design phase từ office) — Site Architect site-focus, may overlap. Khác với Construction Manager (build operation) — Site Architect design + quality focus. Khác với Civil Engineer (structural + system) — different discipline. Khác với Site Engineer (engineering technical) — Site Architect architecture-specific. Hợp tác chặt với: Project Architect (back at office), Construction Manager / General Contractor, Site Engineer (structural, MEP), Foreman + Tradesperson, Owner / Client representative, Inspector + authority, Subcontractor.

  • Construction oversight on site
  • Quality control + design intent
  • RFI + spec clarify
  • Inspection + sign-off coordinate

Site Architect hoạt động construction phase (after design + permit, during build). Báo cáo Project Architect / Principal + dotted line Construction Manager + Owner. Mockup review: pre-construction mockup of key detail (window, facade). Site logistic plan: với contractor, plan delivery + staging. Schedule align: từ design milestone to construction milestone.

3. RFI + change order + coordination

RFI (Request for Information): from contractor when drawing ambiguous or conflict. Site Architect respond + coordinate với Project Architect back at office. Change order evaluate: contractor propose change, evaluate impact on design + cost + schedule. Field decision: minor issue solve on the fly. Major issue escalate: significant deviation discuss với Project Architect + Owner. Substitution review: contractor request material substitute, evaluate equivalence. 03Site Architect cần giỏi gì?

🏗️ Construction process
📐 Drawing reading + spec
📋 Code + regulation
🤝 Communication + diplomacy
🛠️ Material + detail
⚖️ Problem solve + decision
Construction process

Construction sequence end-to-end: foundation, structure, envelope, MEP, finish. Construction method. Building tradition. Read: "Building Construction Illustrated" (Ching), "Construction Project Management" (Sears).

Drawing reading + spec

Read drawing fast + accurate. Cross-reference plan + section + detail. Specifications reading. AutoCAD + Revit + BIM tool. Mobile drawing app on site. Procore / PlanGrid for digital field markup.

Code + regulation

Vietnam: QCVN building code, fire code, structural standard. International: International Building Code (IBC), accessibility (ADA, UK Part M). Zoning. Permit process. Read: "Architect's Studio Handbook".

Communication + diplomacy

Daily multi-party communication — contractor, owner, architect office, inspector. Diplomatic handling disagreement. Authority position needed. Clear written report. Vietnamese + English critical.

Material + detail

Construction material library — concrete, steel, glass, stone, wood, finish. Detail library — junction, transition, waterproofing. Hands-on field experience. Read: "Architectural Detailing" (Allen + Rand).

Problem solve + decision

Field issue need fast solve. Balance design intent + construction reality + cost + schedule. Risk evaluate. Authority to make decision. Calm under pressure.

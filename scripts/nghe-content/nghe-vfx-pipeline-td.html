01Pipeline TD VFX là ai?

Pipeline TD VFX là chuyên gia phát triển software + tool + infrastructure cho VFX studio. Phân loại theo focus: (1) Asset Pipeline TD — asset management, version control, vendor file exchange; (2) Department TD — specific (FX TD, Comp TD, Lookdev TD, Layout TD); (3) Show Pipeline TD — per-show customization (e.g., Mandalorian, Dune); (4) Render Pipeline TD — farm management, cloud render, optimization; (5) USD + Hydra Pipeline TD — Pixar USD framework, RenderMan Hydra delegate; (6) Realtime Pipeline TD — Unreal Engine virtual production (Mandalorian StageCraft style). Phân loại theo company: Major VFX (ILM, Weta, DNEG, Framestore, MPC, Method) — dedicated team; Mid VFX — multi-hat TD; Indie + commercial — generalist.

Khác với Pipeline TD animation (similar but animation studio) — VFX more complex due multi-vendor + live action integration + heavier sim work. Khác với FX TD / Lighting TD / Lookdev TD — those department-specific creative-technical; Pipeline TD infrastructure. Khác với R&D Engineer — Pipeline production-focused; R&D long-term research. Khác với IT — Pipeline CG-tool-specific. Hợp tác chặt với: Pipeline Supervisor / CTO, VFX Supervisor, Production Manager, All department artist + TD, IT team, R&D team, Vendor partner, Production company.

Python + C++ coding

Python — primary VFX language. Master OOP, multithreading, async, packaging, testing. C++ for performance plugin — Houdini SOP / OP, Nuke plugin, Maya plugin. JavaScript / TypeScript for web tool. Bash + PowerShell for scripting.

DCC API (Houdini, Nuke, Maya)

VFX DCC: Houdini HOM + VEX + HDK (C++), Nuke Python + NDK + Blink Script, Maya Python + MEL + Maya API. Plus Katana (lookdev + lighting), Mari (texture), SubstancePainter API, UE5 Python + Blueprint (virtual production).

VFX pipeline knowledge

Hiểu VFX pipeline: plate ingest → tracking → matchmove → layout → animation → FX sim → lighting → comp → final. Each department deeply. Without artist understanding, tool not fit. Ex-artist Pipeline TD often best.

USD + Hydra framework

Modern VFX pipeline standard. Pixar USD for asset interop. Hydra rendering delegate for renderer integration. OpenColorIO for color management. OpenEXR for image. OpenTimelineIO for edit. Adopt by major studio. Read: OpenUSD.org, Pixar tutorial.

System architecture + cloud

Distributed system. Database (PostgreSQL, MongoDB, Cassandra). REST + GraphQL API. Caching (Redis). Message queue (RabbitMQ, Kafka). Cloud (AWS, Azure, GCP) — compute, storage, network. Container (Docker, Kubernetes).

Artist empathy + communication

Tool serve artist creative work. Empathy listen + understand pain. Translate technical → artist language. Patience handle bug report. Documentation thorough.

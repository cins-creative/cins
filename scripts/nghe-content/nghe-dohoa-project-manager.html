Tại agency design như Pentagram, Wieden+Kennedy, Rice Creative VN, Hatcher VN — mỗi project có 1 Project Manager keep machine running smooth. Họ là backbone vận hành creative agency.

Project Manager đồ họa là chuyên gia quản lý project design — từ kick-off đến delivery + invoice. Là bridge giữa client + creative team, manage timeline + budget + scope. Critical role tại agency + design studio + in-house brand team. Without PM, designer disorganized + client unhappy + project late.

01Project Manager đồ họa là ai?

Project Manager (PM) đồ họa là chuyên gia operational + client-facing trong design agency. Phân loại theo company: (1) Branding agency PM — Pentagram, Landor, Rice Creative; (2) Ad agency PM — Wieden+Kennedy, BBDO, Ogilvy, BBH; (3) Digital agency PM — Huge, R/GA, Code & Theory; (4) Design studio PM — Pentagram, MetaDesign; (5) In-house team PM — at brand/company. Phân loại theo title variant: Account Manager (client-relationship focus), Producer (production-execution focus), Project Manager (project-management focus). Khác slight nhưng overlapping role.

Khác với Creative Director (creative leader) — PM operational. Khác với Designer (executor) — PM coordinator. Khác với Account Director (senior client relationship, strategic) — PM more tactical. Khác với Traffic Manager (cross-project resource allocate) — PM single project focus. Hợp tác chặt với: Creative Director, Designer team, Account team, Client, External vendor (printer, photographer, illustrator, freelancer), Studio Manager / Founder.

  • Manage design project
  • Client + creative bridge
  • Timeline + budget + scope
  • Vendor + freelancer coordinate

Vị trí trong quy trình sản xuất

PM đồ họa work tại design agency / studio / in-house team. Báo cáo Account Director / Studio Manager / Founder. Manage 3-15 project simultaneously (smaller scope projects). Per-project: brief → proposal → kick-off → concept → design → revision → approve → production → delivery → invoice. Duration: vary từ 1 tuần (small) đến 6 tháng (rebrand). Daily: status check, client communicate, designer coordinate, budget track.

02Công việc của một Project Manager đồ họa

1. Project kick-off + scope

Receive brief from client hoặc Account team. Scope: define deliverable, timeline, budget. Project plan: phase + milestone + resource. Kick-off meeting với team + client align expectation. Statement of Work (SOW) formalize. Critical for project success — bad scope = entire project struggle.

  • Brief receive
  • Scope define
  • Kick-off + SOW

2. Timeline + budget + day-to-day manage

Timeline track: milestone deadline, daily status, slip flag. Budget track: design hour log, vendor cost, profitability monitor. Daily standups với design team. Client status update: weekly report. Resource coordinate: assign designer, freelancer engage when needed. Risk track: scope creep, resource conflict, vendor delay.

3. Client management + revision

Client primary contact: regular communication, present design, gather feedback. Feedback consolidate: synthesize multiple stakeholder input, present to design team unified. Revision manage: per scope vs out-of-scope (extra cost). Difficult client navigate: misalignment, last-minute change, unrealistic ask. Approval flow: get sign-off at each phase prevent later issue.

  • Client communicate

  • Approval flow

4. Delivery + invoice + post-mortem

Final delivery: coordinate hand-off (file prep, asset organize). Print production coordinate (when applicable) — vendor select, proofing, supervision. Invoice: process + send to client. Project close-out: archive, doc lesson learned. Post-mortem: team retrospective — what worked, what improve. Long-term client relationship: nurture for repeat business.

03Project Manager đồ họa cần giỏi gì?

📋 Project management
🎨 Design knowledge
🤝 Client management
📊 Budget + financial
💬 Communication + diplomacy
🛠️ Tool mastery
Project management

Multiple project parallel. Tool: Asana, Monday, Trello, Basecamp, ClickUp, Notion. Toggl, Harvest for time track. Time-block + prioritize. PMP certification helpful. Read: "The Lean Startup", "Sprint" (Knapp).

Design knowledge

Hiểu design process + craft. Won't be designing nhưng phải hiểu enough — recognize quality, scope effort accurately, communicate intelligently với designer + client. Often PM start as designer.

Client management

Build trust. Manage expectation. Difficult conversation. Diplomatic when push back. Read: "Crucial Conversations" (Patterson), "Getting to Yes" (Fisher).

Budget + financial

Hour rate. Cost vs price. Profitability per project. Vendor invoice. Excel intermediate. Hiểu agency economics — utilization rate, billable, scope creep impact.

Communication + diplomacy

Written + verbal + presentation. Bridge multi-stakeholder. Tone-aware. Documentation. Concise email + clear meeting agenda.

Tool mastery

Project tool above. Plus Figma + Adobe Creative Suite basic (review file, communicate intelligently). Slack, Email, Loom, Zoom. Time-track. CRM (HubSpot, Salesforce for client database).

04Làm cách nào để trở thành Project Manager đồ họa?

1
Học Design / Marketing / Business 3–4 năm

Background: Graphic Design / Communication Design / Marketing / Business / Project Management. VN: ĐH Mỹ thuật Công nghiệp, ĐH FPT Arena, RMIT, ĐH Kinh tế. Quốc tế: Parsons, RISD, SVA, MICA. PMP / PRINCE2 certification valuable.

2
Junior PM / Coordinator / Account Executive 2–4 năm

Entry at agency. VN agency: Rice Creative, Hatcher, Y&R VN, Ogilvy VN, Dentsu VN, Time Universal, Cheil, Phibious. Learn agency operation.

3
Project Manager 4–7 năm

PM role. Manage portfolio of project. Build expertise + reputation.

4
Senior PM / Account Director 7–12 năm

Senior level. Mentor junior PM. Bigger client portfolio. Strategic input.

5
Director of Operations / Studio Director 12+ năm

Top operation role tại agency. Cross-functional leadership.

6
Sự nghiệp Project Manager đồ họa

Long-term path: (1) Director of Operations / COO at agency; (2) Account Director → Partner — at agency; (3) Founder own agency; (4) In-house Head of Design / Brand Operations — at brand; (5) Freelance PM consultant; (6) Crossover to product / tech PM. Lương: VN PM mid 20-45M/tháng; senior 45-90M; Director 90-180M+. International: $60K-$100K mid PM; $100K-$160K senior; Director $160K-$250K+. Agency career path very clear. Plus PM skills transferable rộng — many lateral move possibility.

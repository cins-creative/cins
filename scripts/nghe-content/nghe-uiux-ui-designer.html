Khi Momo app có color hồng tím signature + button radius + icon set consistent feeling polished, khi VinFast app có dark mode automotive premium, khi Tiki search result card có spacing + typography hierarchy clear — đó là tác phẩm của UI Designer.

UI Designer là chuyên gia thiết kế visual layer của digital product — typography, color, spacing, icon, component, micro-interaction. Bridge giữa UX (logic + flow) và visual (look + feel). Critical cho product feeling premium + branded + delightful. Một trong những role hot nhất trong tech industry — Apple, Google, Facebook, Spotify, Netflix design quality phụ thuộc UI Designer.

01UI Designer là ai?

UI Designer là chuyên gia visual design cho digital interface. Phân loại theo platform: (1) Mobile UI (iOS, Android) — biggest demand; (2) Web UI — desktop + responsive; (3) Smart TV / Apple Watch / VR UI; (4) Game UI (HUD, menu); (5) Embedded UI (car dashboard, smart appliance). Phân loại theo focus: (1) Product UI Designer — long-term product; (2) Marketing UI Designer — landing page, campaign; (3) Design System Designer — component library; (4) Brand-focused UI — visual identity bridge. Phân loại theo career: (1) Junior UI Designer; (2) UI Designer; (3) Senior UI Designer; (4) Lead / Principal UI Designer; (5) Design Director. Phân loại theo employer: (1) Tech in-house — VNG, MoMo, VinGroup; (2) Design agency — Lollypop, MOD, Onside; (3) Freelance; (4) Startup founder-designer.

Khác với UX Designer — UX focus problem + flow + IA, UI focus visual + aesthetic + interaction polish. Overlap exist (Product Designer combine both, common at startup). Khác với Visual Designer — broader (print + digital), UI digital + interaction-specific. Khác với Graphic Designer — Graphic broader (print, brand, packaging), UI specifically digital interface. Hợp tác chặt với: UX Designer + Product Designer, Frontend Developer (Web, iOS, Android), Product Manager, Brand Designer, Motion Designer (micro-interaction), QA Designer.

03UI Designer cần giỏi gì?

🎨 Visual design fundamental
✏️ Figma + Sketch + design tool
📐 Typography + grid + spacing
🌈 Color theory + accessibility
⚙️ Design system + token
💻 Platform convention (iOS, Android, Web)
Visual design fundamental

Composition. Hierarchy. Balance. Contrast. Visual rhythm. Read: "Refactoring UI" (Schoger + Wathan), "Designing Interfaces" (Tidwell), "Don't Make Me Think" (Krug).

Figma + Sketch + design tool

Figma industry leader. Sketch Mac legacy. Adobe XD Adobe ecosystem. Auto layout, component, variant, variable in Figma. Prototyping. Plugin for productivity.

Typography + grid + spacing

Type scale (1.125, 1.25 ratio). Modular system (4px / 8px grid). Hierarchy clear. Read: "Thinking with Type" (Lupton), "Grid Systems in Graphic Design" (Müller-Brockmann).

Color theory + accessibility

Color harmony. Brand + semantic + state palette. WCAG AA contrast (4.5:1 text). Dark mode design. Read: "Interaction of Color" (Albers), "Accessibility for Everyone" (Kalbag).

Design system + token

Component library architecture. Token (color, spacing, type) system. Naming convention. Read: "Atomic Design" (Frost), "Design Systems" (Suarez et al.).

Platform convention (iOS, Android, Web)

iOS Human Interface Guidelines. Android Material Design. Web responsive. Platform-specific gesture + navigation pattern.

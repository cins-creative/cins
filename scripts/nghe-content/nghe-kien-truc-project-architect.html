Tại firm như Foster + Partners, Gensler, BIG, HOK, hay tại VN với firm như Vo Trong Nghia, 1+1>2 — một Principal Architect có vision concept, nhưng một Project Architect là người lead specific project đến completion. Họ là backbone của firm architectural.

Project Architect là chuyên gia lead specific project — từ design development đến construction completion. Là role mid-senior architect responsible for project execution, client face, team management cho 1 project specific. Bridge giữa Principal vision + project reality.

01Project Architect là ai?

Project Architect là kiến trúc sư senior chịu trách nhiệm toàn bộ về 1 specific project. Phân loại theo phase: (1) Schematic Design (SD) + (2) Design Development (DD) + (3) Construction Documents (CD) + (4) Bidding + Negotiation + (5) Construction Administration (CA). Project Architect lead all or some phase. Phân loại theo project type: Residential, Commercial / office, Retail / hospitality, Civic / institutional, Industrial, Cultural (museum, theater), Mixed-use. Phân loại theo firm: Boutique firm (Project Architect may = Principal), Large firm (clear hierarchy: PA reports Principal).

Khác với Principal Architect (firm leadership, design oversight cross-project) — Project Architect single project leader. Khác với Design Architect (concept + design focus) — Project Architect execution + management. Khác với Job Captain (junior senior — handles CD detail) — Project Architect more senior. Khác với Architectural Designer (entry/mid) — Project Architect senior project lead. Hợp tác chặt với: Principal Architect (upward report), Design Architect, Job Captain + Designer (team under), Engineer (structural, MEP), Contractor, Consultant (landscape, lighting, acoustic), Client.

  • Lead specific project
  • Client + consultant coordinate
  • Team management project
  • Quality + spec oversight

Vị trí trong quy trình sản xuất

Project Architect work tại architectural firm. Báo cáo Principal Architect / Partner / Studio Director. Manage team: 2-8 architect / designer / job captain. Lead 1-3 project simultaneously. Project duration 1-5 năm typical. Project total fee $500K-$50M+ depending on project size. Mix of office + site visit + client meeting work.

02Công việc của một Project Architect

1. Design development + technical design

From Principal's concept → develop into detailed design. Schematic Design (SD): refine massing, plan, key element. Design Development (DD): detail material, system (structural concept, MEP coordinate), envelope. Construction Documents (CD): full technical drawing + specification — what contractor build from. Review junior designer work. Make design decision daily — balance creative intent + practical constraint.

  • SD → DD → CD
  • Design decision
  • Review junior

03Project Architect cần giỏi gì?

📐 Technical design + detailing
📋 Project management
🤝 Client management
🛠️ Construction know-how
🏗️ BIM + Revit mastery
⚖️ Code + regulation
Technical design + detailing

Deep architectural detail knowledge. Construction detail, envelope, structural integration, MEP coordinate. Material specification. Read: "Architectural Graphic Standards", "Building Construction Illustrated" (Ching).

Project management

Schedule + fee + team manage. Tool: Microsoft Project, Smartsheet, Asana, BQE Core, Newforma. Hiểu architectural project lifecycle (SD-DD-CD-CA). PMP helpful but architecture-specific PM certification (AIA PM) also available.

Client management

Primary client face — communication, expectation, decision facilitation. Manage difficult client. Present clearly. Read: "Crucial Conversations" (Patterson), "Never Split the Difference" (Voss).

Construction know-how

Hiểu construction process + contractor mindset + trade work. Site experience critical. Read: "Means & Methods of Construction", AIA contract document.

BIM + Revit mastery

Revit primary tool. Rhino + Grasshopper for complex form. SketchUp for concept. AutoCAD still common. Navisworks for clash detection. BIM 360 / ACC for collaboration. Adobe Creative Suite for present.

Code + regulation

Building code — VN: QCVN, TCVN; US: IBC, ADA, energy code. Fire code. Accessibility. Zoning. Local jurisdiction know. Sustainability certification (LEED, LOTUS VN). Plus professional ethics for architect license.

04Làm cách nào để trở thành Project Architect?

1
Học Kiến trúc 5 năm (B.Arch)

VN: ĐH Kiến trúc Hà Nội (HAU), ĐH Kiến trúc TP.HCM, ĐH Xây dựng (NUCE), ĐH Bách khoa Đà Nẵng, RMIT. International: Harvard GSD, Yale SOA, MIT, UCL Bartlett, AA London, Cooper Union, USC, RISD, ETH Zurich, TU Delft. M.Arch optional but valuable cho academic + competitive firm.

2
Architectural Designer 2–4 năm

Entry post-graduation. Document drafting, design assist. VN: Vo Trong Nghia, 1+1>2, Sunlim, AIA VN, NDA, Studio Anettai; large firm Gensler VN, AECOM, Mode. Get architect license: VN Chứng chỉ hành nghề kiến trúc (after 5 năm); US ARE exam + state license.

3
Job Captain 4–7 năm

Senior designer + sub-team lead. CD detail expertise. Mentor junior. Path to PA.

4
Project Architect 7–12 năm

PA role. Lead project end-to-end. Mid-sized project first → larger.

5
Senior Project Architect / Associate 12–17 năm

Multiple complex project. Mentor PA team. Track to Associate / Senior Associate.

6
Sự nghiệp Project Architect

Long-term path: (1) Principal Architect / Partner — firm leadership; (2) Studio Director — large firm; (3) Own firm founder; (4) Construction management / owner-side — switch to client representation; (5) Real estate development — architect-developer hybrid; (6) Academic + research; (7) Specialty consulting (sustainability, acoustics, healthcare specialty). Lương: VN PA mid 25-50M/tháng; senior 50-100M; Principal 100-300M+; partner equity earning. US: $80K-$130K mid PA; $130K-$200K senior; Principal $200K-$400K+; partner equity. Architecture career long-term (30-40 năm typical). Reputation + portfolio compound. License + experience valuable lifetime asset.

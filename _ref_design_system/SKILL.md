If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

## Quick orientation

- **Brand:** CINs — connection platform for Vietnamese visual creative industries (Film, Game, Cinema, Architecture, Fashion, Design)
- **Primary user:** Vietnamese high-school students (THPT)
- **Tone:** friendly, encouraging, casual "bạn"; sentence case everywhere; no emoji
- **Foundation file:** `colors_and_type.css` — import this at the top of any new HTML
- **UI components:** `ui_kits/web/` — JSX component recreations + an interactive demo
- **Brand assets:** `assets/` — logo, mascots, illustrations, patterns
- **Hard rules:** rounded corners only (never sharp); logo only on white; no gradients; no emoji
